Sign in with GitHub
What's this? By clicking this button, you're telling GitHub that you grant the Electric Book Manager permission to connect to your account. If you don't have an account, you'll be prompted to create one for free. Once you've done that, you'll see your dashboard here.

Create and edit books as websites and PDFs

  • Create and contribute to Electric Book projects
    Create a new book project at the click of a button, or join someone else’s project just by entering its GitHub name. Your work stays in your own copy of the project.
  • Edit your books and view them as websites and PDFs
    Edit your books using plain-text markdown, and output a website version, a print-ready PDF, or a screen PDF instantly.
  • Submit and review changes with version control
    When contributing to someone else’s project, submit your changes for them to review. And review others’ contributions to your projects. Accept, reject or modify those contributions line by line.
  • Sync everything safely in your GitHub account
    When you commit your changes to version control, everything syncs with your GitHub account. You get all the peace of mind that the world’s largest code host provides.
